# Cognetix-Python-Internship--stage2

### Simple File Encryptor
This is a Python-based project that uses the `cryptography` library to encrypt and decrypt text files securely.

## How to use:
1. Run `python encryptor.py`.
2. Follow the terminal prompts to choose 'E' (Encrypt) or 'D' (Decrypt).
3. Enter the filename to perform the operation.
